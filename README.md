Небольшое веб-приложение на Python и Flask для быстрой проверки HTTP(S)‑прокси:

- **Проверка доступности**: удаётся ли выполнить запрос через прокси.
- **Время отклика**: сколько миллисекунд занимает ответ.
- **Видимый IP**: какой IP видит внешняя служба (через `https://httpbin.org/ip`).

Интерфейс у меня получился простым XD: одно поле для прокси и таймаут.

## Установка

Требуется: **Python 3.10+**.

```bash
cd c:\Users\User\Desktop\CheckProxy
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
cd c:\Users\User\Desktop\CheckProxy
.venv\Scripts\activate
python app.py
```

После запуска Вы можете открыть в браузере:

- `http://127.0.0.1:5000`

## Формат прокси

Поддерживаются стандартные HTTP(S)‑прокси:

- Без авторизации: `host:port` или `http://host:port`
- С авторизацией: `http://user:password@host:port`

Приложение делает запрос к `https://httpbin.org/ip` через указанный прокси и выводит:

- Статус — работает прокси или нет.
- HTTP‑код ответа (если есть).
- Время отклика в миллисекундах.
- IP‑адрес, который виден снаружи.
